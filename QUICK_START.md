# 🚀 빠른 시작 가이드

## ✅ 좋은 소식!

터미널에 오류 메시지가 보여도 **서버는 정상적으로 실행 중입니다!**

터미널에 `GET / 200` 같은 메시지가 보이면 = 웹사이트가 작동하고 있다는 뜻입니다!

---

## 🌐 웹사이트 확인하기

1. 웹 브라우저를 엽니다 (Chrome, Edge 등)
2. 주소창에 입력: `http://localhost:3000`
3. Enter 키 누르기

**인트로 화면이 보이면 성공입니다!** 🎉

---

## ⚠️ 오류 메시지에 대해

터미널에 보이는 경고(⚠) 메시지들은:
- **경고일 뿐, 사이트는 정상 작동합니다**
- 방금 수정했으니 이제 사라질 것입니다
- 빨간색 오류(⨯)만 문제입니다 (이것도 수정했습니다)

---

## 🔧 다음 단계

웹사이트가 보이면:

1. **인트로 화면** → "환영합니다" 버튼 클릭
2. **메인 화면** → "전통혼례" 또는 "돌잔치" 선택
3. **신청 프로세스** 테스트

---

## 🛑 서버 중지하기

서버를 멈추려면:
- 터미널에서 `Ctrl + C` 키 누르기

---

## 📝 문제가 있나요?

웹사이트가 안 보이면:
1. 터미널에 `Ready in` 메시지가 있는지 확인
2. 브라우저 주소가 정확한지 확인 (`http://localhost:3000`)
3. 다른 프로그램이 3000번 포트를 사용 중일 수 있음

---

**지금 브라우저에서 `http://localhost:3000`을 열어보세요!** 🚀
